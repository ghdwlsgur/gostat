# Overview

An Interactive CLI Tool that diagnoses whether certificates are applied with edge servers(a record address) as proxy.

# Installation

### homebrew

```bash

# [install]
brew tap ghdwlsgur/gostat
brew install gostat

# [upgrade]
brew upgrade gostat
```
